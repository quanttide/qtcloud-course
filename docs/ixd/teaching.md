# 教学管理

## 设计原则

### 班级为中心

所有教学管理操作围绕班级展开。查看班级状态、管理学员、跟踪进度、调整教学计划都在班级上下文中完成。

### 状态可见

每个班级的生命周期状态（筹备中/进行中/已结束）在列表和详情中清晰展示。关键变化（开课、结课）需要确认操作。

### 引用不复制

班级引用课程研发的 Program/Course/Lesson 内容，不重新定义。班级对教学内容的调整以覆盖形式存在，不影响原始内容。

## 核心元素

### 班级列表

- 卡片式列表展示所有班级，每张卡片包含：班级名称、引用专业/课程、教学周期、状态标签
- 卡片右侧展示核心指标：学员数、进度条（百分比）、状态（筹备中/进行中/已结束）
- 顶部筛选标签：全部、筹备中、进行中、已结束。支持搜索框按名称搜索
- 点击卡片选中，下方展开班级详情面板

### 班级详情面板

- 选中班级后在列表下方展开，不跳转页面。同一时间只能选中一个班级
- 顶部展示四格指标卡片：学员数、出勤率、完成率、平均分
- 下方是标签导航：教学计划、学员、进度、设置

### 教学计划标签

- 教学日历表格：周次 → 日期 → 课时内容 → 状态（完成/进行中/待开始）
- 每一行显示课时名称、对应的 Lesson 来源、授课方式（线上/线下）、授课教师
- 点击行展开课时详情，包含教学素材链接、学员完成情况统计
- 顶部有"[调整计划]"按钮，进入编辑模式后可拖拽调整课时顺序、修改日期、替换课时

### 学员标签

- 学员列表表格：姓名、学号、出勤率、完成率、综合成绩
- 支持按出勤率和完成率排序，搜索框支持按姓名搜索
- 点击学员打开侧边面板查看详情：个人信息、各课时完成状态、成绩记录
- 顶部有批量操作：[导出名单] [导入成绩]

### 进度标签

- 课程维度的教学进度总览。以引用的 Program/Course 结构展示各课程的完成比例
- 每个课程行显示：课程名、总课时数、已完成课时数、完成率、平均出勤率
- 点击课程展开课时级别的完成详情，展示每个课时的学员完成人数

### 设置标签

- 班级基本信息编辑：名称、教学周期、引用内容（只读，显示引用路径）
- 授课教师管理：添加/移除教师，每位教师显示姓名和负责内容范围
- 状态操作：[开课] / [结课] 按钮，点击后需二次确认。开课后教学计划不可大规模调整，结课后学员成绩冻结

### 新建班级

- 点击 [+ 新建班级] 弹出创建向导
- 第一步：选择引用的专业或课程（从课程研发的已有内容中选择）
- 第二步：填写班级名称、选择教学周期（起止日期）
- 第三步：设置授课教师和初始学员名单（支持 CSV 导入）
- 确认后创建班级状态为"筹备中"，跳转到班级详情页

## 交互行为

- 班级列表点击选中/取消选中，同时只能展开一个班级详情。选中切换时详情面板内容平滑过渡
- 标签导航切换时不刷新页面，只切换内容区域
- 教学日历的课时行点击展开/收起详情，不跳转
- 调整教学计划时，如果引用的 Lesson 在课程研发侧发生了更新，班级侧显示更新标记，提示"课程内容已更新，是否同步？"
- 开课操作后，教学计划进入锁定状态，调整需要申请审批

## 交互流程

### 查看班级教学进度

用户进入教学管理 → 班级列表展示所有班级 → 点击"浙理班级" → 下方展开详情面板 → 默认展示教学计划标签 → 查看教学日历 → 点击进行中的课时行展开详情 → 查看该课时的学员完成情况

### 调整教学计划

用户在班级详情 → 点击教学计划标签 → 点击"调整计划" → 教学日历进入编辑模式 → 拖拽课时行调整顺序 → 修改某个课时的日期 → 点击保存 → 系统校验日期冲突 → 保存成功退出编辑模式

### 管理学员

用户进入班级详情 → 点击学员标签 → 查看学员列表 → 搜索框输入姓名定位学员 → 点击学员打开侧边面板 → 查看学员各课时完成状态 → 可关闭侧边面板继续浏览列表

### 开课操作

用户在班级详情 → 点击设置标签 → 确认班级信息和教师已配置 → 点击"开课"按钮 → 弹出确认对话框提示"开课后教学计划仅可微调" → 确认 → 班级状态更新为"进行中" → 学员标签可开始记录出勤和成绩 → 教学计划标签锁定编辑
