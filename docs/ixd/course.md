# 课程研发

## 设计原则

### 树形导航

专业 → 课程 → 课时形成三级树形结构。用户在树中定位到层级后，右侧展示对应详情，不跳转页面。

### 内容与结构分离

左侧维护层级结构（增删改排序），右侧编辑内容。结构调整不影响内容，内容编辑独立于结构位置。

### 渐进式展开

默认只展示专业级别，逐级展开。不一次性加载全部课时，按需加载减少信息过载。

## 核心元素

### 课程结构树

- 左侧面板宽度固定 280px，支持拖拽调整
- 树分三级：专业（📁）→ 课程（📁）→ 课时（📄），每级前有展开/折叠图标
- 每个节点右侧显示状态标记：已发布（绿色圆点）、待发布（灰色圆点）、草稿（黄色圆点）
- 右键点击节点弹出上下文菜单：编辑、重命名、新建子级、上移、下移、删除
- 拖拽节点可调整排序，也可将课程拖到其他专业下
- 树顶有搜索输入框，输入后过滤匹配的节点，匹配项高亮

### 内容面板

- 右侧面板根据树中选中的层级展示不同内容
- 专业视图：展示基本信息（名称、描述、状态、创建时间）和包含的课程列表
- 课程视图：展示课程基本信息、课时列表（表格形式，包含课时名、时长、状态）、版本历史
- 课时视图：展示课时内容编辑器，包含标题、教学目标、教学内容（富文本）、素材附件列表、AI 生成内容

### 详情编辑

- 点击编辑按钮，字段变为可编辑状态。专业和课程信息是表单（文本输入、下拉选择），课时内容是富文本编辑器
- 编辑时底部出现操作栏：[保存] [取消] [保存为草稿]
- 未保存时树节点上显示未保存标记（红色圆点），离开时提示确认

### 新建向导

- 点击 [+ 新建专业/课程/课时] 按钮弹出分步向导
- 新建专业：填写名称、描述、选择图标
- 新建课程：选择所属专业、填写名称、描述、设置课时数量模板
- 新建课时：选择所属课程、填写标题、设置预估时长
- 向导最后一步展示摘要确认，确认后在树中对应位置插入新节点并选中

## 交互行为

- 树节点点击切换内容面板，不刷新页面。切换动画 200ms
- 树节点拖拽时显示插入位置指示线，松手后自动保存排序。如果拖拽跨专业，需确认"移动课程会断开与原专业的关系"
- 右键上下文菜单位置跟随鼠标，点击菜单外部关闭
- 富文本编辑器支持 Markdown 快捷输入和粘贴图片（自动上传）
- AI 生成内容用特殊背景色和标记区分，用户可接受、修改或拒绝。拒绝后 AI 根据反馈重新生成

## 交互流程

### 浏览课程结构

用户进入课程研发 → 左侧树默认展开第一级专业 → 点击专业展开课程列表 → 点击课程展开课时列表 → 点击课时节点 → 右侧展示课时内容 → 可继续浏览其他节点

### 新建课时

用户在树中右键课程节点 → 选择新建课时 → 右侧弹出新建向导 → 填写课时标题和时长 → 确认 → 树中课程下新增课时节点并选中 → 内容面板切换到课时编辑器 → 开始编写内容

### 调整课程结构

用户拖拽课时节点到新位置 → 插入位置显示指示线 → 松手 → 系统保存新排序 → 树节点更新序号 → 右侧内容面板不变

### 使用 AI 生成内容

用户在课时编辑器 → 点击 AI 生成按钮 → 选择生成类型（教学目标/教学内容/练习题） → 输入提示词 → 点击生成 → AI 生成内容展示在编辑器（特殊标记） → 用户审阅 → 接受（内容正常化）或修改后保存 / 拒绝（AI 重新生成）
