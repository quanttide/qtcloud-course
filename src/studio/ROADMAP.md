# ROADMAP — Studio

## v0.0.4 — 底部导航 → 侧边导航

**现状**：底部 `NavigationBar` 仅支持 3 个 tab，数量受限，不符合 B 端产品习惯。

**目标**：改为左侧导航栏（Sidebar）布局，方便未来扩展导航项，与 B 端产品设计语言对齐。

### 布局

```
┌─────────────────────────────────────────┐
│  AppBar                                  │
├──────────┬──────────────────────────────┤
│ Sidebar  │                              │
│          │      Content Area             │
│ 🏠 仪表盘 │      (IndexedStack)           │
│ 📖 课程研发│                              │
│ 👥 教学管理│                              │
│          │                              │
│ ➕ 更多.. │                              │
└──────────┴──────────────────────────────┘
```

全屏页面（如 PreviewScreen）隐藏 Sidebar，由路由统一控制。

### 要点

- 固定 240px，v0.0.4 不做折叠/响应式
- 图标用 `Icons.*`，与现有 Material Design 一致
- 当前导航项蓝色高亮
- 路由状态区分 `tab` / `preview(lessonId)`，`PreviewScreen` 不再自建 `Scaffold`
