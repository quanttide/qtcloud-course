# ROADMAP — Studio

## 底部导航 → 侧边导航

**现状**：底部 `NavigationBar` 仅支持 3 个 tab（仪表盘 / 课程研发 / 教学管理），数量受限，不符合 B 端产品习惯。

**目标**：改为左侧导航栏（Sidebar）布局，方便未来扩展更多导航项，与 B 端产品设计语言对齐。

### 布局

```
┌─────────────────────────────────────────┐
│  AppBar                                  │
├──────────┬──────────────────────────────┤
│  Sidebar  │                              │
│           │      Content Area            │
│  📊 仪表盘 │                              │
│  📖 课程研发│      (IndexedStack)          │
│  👥 教学管理│                              │
│           │                              │
│  ➕ 更多.. │                              │
├──────────┴──────────────────────────────┤
│  Status bar (可选)                       │
└─────────────────────────────────────────┘
```

### 设计要点

- **Sidebar 宽度**：240px（标准 B 端侧栏宽度）
- **图标 + 文字**：每个导航项带 icon 和 label
- **当前项高亮**：蓝色背景或左侧蓝色竖线
- **可折叠**：支持 `Collapsed` 状态（仅图标，64px），腾出内容空间
- **响应式**：窗口缩小时自动折叠为图标模式
- **扩展性**：新增导航项只需在列表中添加一项，无需调整布局
